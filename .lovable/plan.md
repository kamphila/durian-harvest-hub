# 🍈 ระบบล้งทุเรียน (Durian Packing House)

## ภาพรวม

ระบบบริหารจัดการล้งทุเรียนครบวงจร ตั้งแต่รับซื้อ → คัดเกรด → แพ็ค → ส่งออก พร้อม Dashboard วิเคราะห์กำไร Real-time ภาษาไทยทั้งระบบ ธีม AdminLTE สีเขียวเข้มเหมือนเปลือกทุเรียน

รองรับหลาย  ๆ บริษัท 1 Lgin สังกัดได้ 1 บริษัทเท่านั้น

**เริ่มต้นด้วย UI + Mock Data ก่อน** แล้วค่อยต่อ Database ทีหลัง

## เฟส 2: Transaction - เอกสารหลัก

### 2.1 รับซื้อ (Purchase Intake)

- รายการรับซื้อ: วันที่, เลขที่, ทะเบียนรถ, สายตัด, ผู้จำหน่าย
- ฟอร์มรับซื้อ: เลือกผู้จำหน่าย/สายตัด, บันทึกสินค้า/เกรด/น้ำหนักแยกเข่ง
- คำนวณน้ำหนัก + คิดเงินอัตโนมัติ + หักมัดจำ
- จ่ายชำระ: โอน/เงินสด + เลือกบัญชี
- พิมพ์ใบจ่ายเงิน (Payment)

### 2.2 สัญญาซื้อเหมาสวน

- รายการสัญญา + ฟอร์ม: วันที่, เลขที่, ชื่อสวน, มูลค่าเหมา
- จ่ายชำระ: โอน/เงินสด

### 2.3 จ่ายมัดจำล่วงหน้า

- อ้างอิงสัญญา, สมาชิก, สินค้า, เกรด, น้ำหนัก, ค่าตัด/กก.

### 2.4 แพ็คกิ้ง (Packing)

- บันทึกบรรจุกล่อง: สินค้า, จำนวน, หน่วยนับ, ราคา
- พิมพ์ QR Code / Lot No.
- Traceability: ย้อนกลับดูได้ว่ากล่องนี้มาจากสวนไหน
- บันทึกผลคัดเกรด + แยก Size + คำนวณ Yield %

### 2.5 ขาย/ส่งออก (Sales/Export)

- รายการขาย: วันที่, เลขที่, เลขตู้ Container, ลูกค้า, มูลค่า
- พิมพ์เอกสาร: Invoice, Packing List, ใบรับรอง GAP, Phytosanitary, ใบปิดตู้
- รับชำระ: หักมัดจำ/โอน

### 2.6 รับมัดจำล่วงหน้า (จากลูกค้า)

- อ้างอิงสัญญา, สินค้า, เกรด, จำนวน

### 2.7 บันทึกค่าใช้จ่าย

- ฟอร์มบันทึก + พิมพ์ใบสำคัญจ่าย

### 2.8 คลังสินค้า

- Stock ดิบ / คัดแล้ว / แพ็คแล้ว
- ตัด Stock อัตโนมัติเมื่อมี Transaction

## เฟส 3: Dashboard & รายงาน

### 3.1 Executive Dashboard

- ปริมาณรับซื้อวันนี้ (ตัน, จำนวน Lot, เปรียบเทียบ)
- กำไรขั้นต้น (วันนี้/เดือนนี้/ต่อ Container)
- Yield % (ส่งออก/ตกเกรด/สูญเสีย)
- ราคาซื้อเฉลี่ย vs ราคาขาย + Margin
- Container Status (เตรียม/โหลด/ส่งออก/รอรับเงิน)
- กราฟ Trend: รับซื้อ, Yield, กำไรต่อ Container

### 3.2 Dashboard ฝ่ายรับซื้อ

- Top 10 สวน, ราคาซื้อเฉลี่ยรายสวน, Yield รายสวน, กำไรต่อสวน

### 3.3 Dashboard ฝ่ายคัดเกรด

- Yield % รายวัน/ต่อ Lot/ต่อพนักงาน, สัดส่วนเกรด

### 3.4 Dashboard คลังสินค้า

- Stock คงเหลือแต่ละประเภท, อายุสินค้า (Day Aging), Lot ใกล้ครบกำหนด

### 3.5 Dashboard ฝ่ายขาย

- ยอดขายรายลูกค้า/ประเทศ, กำไรต่อ Shipment, ลูกหนี้คงค้าง

### 3.6 Dashboard วิเคราะห์กำไร

- กำไรต่อ Lot / Container / สวน / เกรด

### 3.7 ระบบแจ้งเตือน (Alert)

- Yield ต่ำกว่า 65%, Loss สูงผิดปกติ, ราคาซื้อเกิน Margin, ลูกค้าค้างชำระ, Container ใกล้ส่งออก

### 3.8 รายงานการเงิน

- จ่ายเงินสวน, รับเงินลูกค้า, ลูกหนี้/เจ้าหนี้, กำไรขาดทุนต่อ Lot, สัญญาซื้อเหมา
